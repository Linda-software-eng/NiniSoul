# NiniSoul

NiniSoul is an Android application designed to simplify meal planning and reduce food waste. It suggests recipes based on the ingredients you already have in your kitchen, making home cooking more convenient and sustainable.

## Features
* **Fridge-based Suggestions:** Enter what ingredients you have, and get recipe recommendations instantly.
* **Digital Recipe Library:** Store and organize your favorite family recipes in one place.
* **Meal Planning:** Plan your meals to stay organized throughout the week.
* **User Profile:** Keep track of your cooking history and preferences.

## Tech Stack
* **Language:** Kotlin
* **Framework:** Android SDK
* **Architecture:** MVVM (Model-View-ViewModel)

## How to Run
1. Clone this repository to your machine.
2. Open the project in **Android Studio**.
3. Sync the project with Gradle files.
4. Run the app on an emulator or physical Android device.
